# 🤖 Telegram AI Bot

A powerful Telegram bot with support for multiple AI providers including OpenAI GPT, Google Gemini, and OpenRouter (LLaMA, Claude, etc.).

## Features

- 🔄 Multiple AI Providers (OpenAI, Gemini, OpenRouter)
- 💾 Conversation memory (remembers context)
- 🎯 Model switching without restarting
- ⚡ Fast, async responses
- 🔒 Optional access control
- 📊 Status monitoring
- 💬 Natural conversation flow
